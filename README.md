**Author:** Davide Marelli

**Status:** Completed

## 🎯 Project Objective
This project bridges the gap between **Information Technology** and **Business Administration**. I designed a relational database simulating a real-world e-commerce environment to demonstrate how raw data can be transformed into actionable business insights.

## 🛠️ Tech Stack & Skills Used
* **Language:** SQL
* **Techniques:** Database Creation, Data Insertion, Joins, Aggregation Functions (`SUM`, `COUNT`), Grouping, Date Extraction.
* **Business Concepts:** KPI tracking, Customer Lifetime Value (LTV), Revenue Analysis.

## 📈 Business Questions Answered
In the `ecommerce_business_analysis.sql` file, I wrote queries to solve real company problems:
1. **Total Revenue by Month:** How is the company performing over time?
2. **Customer Lifetime Value (LTV):** Who are our most valuable/VIP customers?
3. **Geographic Sales Analysis:** Which countries generate the most revenue?
